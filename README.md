# loftschool-shop
Страница интернет-магазина

![demo](demo.png)

### Установка
```
git clone https://github.com/AndreyBalandin/loftschool-shop.git site
cd site
npm install
bower install
```
### Сборка html-файлов из jade-шаблонов
```
gulp jade
```
### Сборка sass-файлов из scss-шаблонов
```
gulp sass
```
### Сборка всего проекта в каталог dist
```
gulp build
```
